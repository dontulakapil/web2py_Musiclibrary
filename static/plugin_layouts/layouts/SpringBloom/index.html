Welcome To Spring Bloom

This template is just what came together after browsing Open Stock Photography. Whilst searching through their stocks I came across the image I later used in the header & navigation area of this template. A few specifics, browser-based font resizing is fully supported and images have been optimised for size and loading speed. Other than that, this design maintains my usage of using valid semantic and well-structured XHTML 1.0 Strict and CSS 2 code.
