About This Design

This design was inspired by a site I ran across (www.peachcollection.com). I found the design very appealing. I decided to create my interpretation of it from scratch using full standards compliance.

One image is used in this layout. I have repeated the background image in different spots to show possible placement of graphics.

This design validates to XHTML 1.0 Strict, uses CSS for layout, contains no hacks, and include skip links for accessibility. It has been tested in Firefox, IE 6 and IE 7 Beta 2, Opera, Konqueror, and Safari.
