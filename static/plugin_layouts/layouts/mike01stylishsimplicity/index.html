Open Source Template

This template is open source web-design, which means that you can use it in any way you like. I just kindly ask that you leave the credit text and link in the footer (the one which says "Design by Mike Yarmish"). I would really appreciate such support.

Design Features

The design is fully liquid. In 800px it works as fine as in 1024px and 1280px. If you want to change columns' places you could edit float and text-align properties in CSS.
